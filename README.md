# Textnet Excalibur

This is an Excalibur-based prototype of the TXT NET

    npm install
    npm run dev

Proper distro will be done in Electron later.

# CONTROLS

- Movement: UP, DOWN, LEFT, RIGHT
- Artifacts:
    - Push:  (move towards artifact)
    - Pickup: SHIFT + (push)
    - Put down: SHIFT
    - Enter: CTRL + (push)
    - Leave: ESC 
- Written Word:
    - Enter: CTRL+ENTER
    - Leave: CTRL+ENTER
- Spoken Word:
    - Say: ENTER
